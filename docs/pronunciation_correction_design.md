# 英语发音纠错系统设计方案

## 1. 当前系统分析

### 1.1 现有功能
- 基于myprosody库的韵律特征分析
- WhisperASR语音识别（支持中英文）
- 基础发音评分（流畅度、发音准确性、韵律表现、内容准确性）
- 简单的文本匹配评估

### 1.2 主要局限性
- **缺乏音素级别分析**：无法识别具体的元音、辅音发音错误
- **共振峰分析不足**：无法精确测量元音的共振峰频率
- **音标对比缺失**：无法提供IPA级别的发音纠错
- **发音错误定位不准确**：只能给出整体建议，无法定位具体错误位置
- **缺乏个性化纠错**：无法根据学习者母语背景提供针对性建议

### 2.2 核心模块设计

#### 2.2.1 音素分析模块 (Phoneme Analysis)
**技术选型**: Parselmouth + Praat

**功能**:
- 音素分割和识别
- 元音共振峰分析 (F1, F2, F3)
- 辅音频谱特征分析
- 音素时长分析
- 音素边界检测

**实现方案**:
```python
class PhonemeAnalyzer:
    def __init__(self):
        self.praat = parselmouth.praat
        
    def extract_formants(self, audio_path, start_time, end_time):
        """提取指定时间段的共振峰"""
        
    def segment_phonemes(self, audio_path, transcript):
        """音素分割"""
        
    def analyze_vowel_quality(self, formants):
        """分析元音质量"""
        
    def analyze_consonant_features(self, audio_segment):
        """分析辅音特征"""
```

#### 2.2.2 发音错误检测模块 (Error Detection)

**错误类型分类**:
1. **元音错误**
   - 共振峰偏移 (F1/F2不准确)
   - 元音长度错误
   - 双元音滑动不准确

2. **辅音错误**
   - 爆破音释放不完整
   - 摩擦音频谱特征异常
   - 鼻音共振异常
   - 流音发音位置错误

3. **韵律错误**
   - 重音位置错误
   - 语调模式不当
   - 节奏不规律
   - 停顿位置不合适

**实现方案**:
```python
class PronunciationErrorDetector:
    def __init__(self):
        self.vowel_standards = self._load_vowel_standards()
        self.consonant_patterns = self._load_consonant_patterns()
        
    def detect_vowel_errors(self, formants, target_phoneme):
        """检测元音发音错误"""
        
    def detect_consonant_errors(self, spectrum, target_phoneme):
        """检测辅音发音错误"""
        
    def detect_prosody_errors(self, pitch_contour, stress_pattern):
        """检测韵律错误"""
```

#### 2.2.3 智能纠错建议模块 (Correction Guidance)

**建议类型**:
1. **具体发音指导**
   - 舌位调整建议
   - 口型描述
   - 气流控制指导

2. **练习建议**
   - 最小对比练习
   - 渐进式练习序列
   - 相似音素对比

3. **视觉辅助**
   - 共振峰图表对比
   - 音调曲线显示
   - 音素时长可视化

## 6. 风险与挑战

### 6.1 技术风险
- Parselmouth库的稳定性和性能
- 不同口音和方言的适应性
- 背景噪音对分析精度的影响

### 6.2 解决方案
- 建立多样化的测试数据集
- 实现鲁棒的噪音过滤算法
- 提供多种分析模式选择

## 7. 总结

本方案通过集成专业的语音分析工具，建立完整的音素级别发音纠错系统，能够：

1. **精确定位发音错误**：从音素级别识别具体问题
2. **提供专业指导**：基于语音学原理的科学建议
3. **个性化学习**：根据学习者背景定制纠错策略
4. **可视化反馈**：直观的图表和数据展示
5. **渐进式改善**：系统性的练习推荐

这将显著提升英语发音学习的效果和用户体验。
